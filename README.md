# 🌞 Solaris
### Free AI workspace for everyone

Solaris is an open-source AI workspace — free forever.

- 🤖 AI assistant 24/7
- 📧 Gmail, Calendar, Telegram, WhatsApp  
- 🎨 Free image generation — funded by ads
- 🎬 Free video generation — watch ads, get videos
- ☁️ Cloud-powered — no GPU required
- 🌍 Ad revenue funds schools and humanitarian aid

## Quick Start
git clone https://github.com/Myerss00/Solaris.git
cd Solaris
cp .env.example .env
docker compose up -d --build
